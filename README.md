#### 插件介绍
加载插件后，页面左右下角出现拉姆和雷姆，点击就会触发至顶和至底功能。

#### 注意事项
 - 本插件需要加载 JQuery，因为有的主题模板本身已经引入加载JQuery，所以插件设置内可以选择是否禁止插件加载自身的JQuery。
 - 如开启插件后没有出现心爱的雷姆和拉姆，请检查使用的主题模板的页首钩子`<?php $this->header(); ?>`和页脚钩子`<?php $this->footer(); ?>`是否存在。
