加載插件後，頁面左右下角出現拉姆和雷姆，點擊就會觸發至頂和至底功能  

如開啟插件後沒有出現心愛的雷姆和拉姆，請檢查使用的外觀是否存在頁首鉤子`<?php $this->header(); ?>`和頁腳鉤子`<?php $this->footer(); ?>`

> 插件自動加載`JQuery`，如果外觀本身已加載，請至插件設置內禁止自動加載
